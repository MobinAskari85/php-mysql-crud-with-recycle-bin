# PHP MySQL CRUD System with Recycle Bin

A full-stack CRUD (Create, Read, Update, Delete) web application developed using **PHP**, **MySQL**, **jQuery AJAX**, and **Bootstrap 5**.

This project includes advanced data management features such as soft delete (Recycle Bin), restore functionality, sorting, filtering, and dynamic UI updates without page reload.

---

## 🚀 Features

- Create new users dynamically
- Read and display data from MySQL database
- Update user information inline
- Soft delete system (Recycle Bin)
- Restore deleted records
- Permanent delete option
- AJAX-based operations (without page refresh)
- Sorting by name and turn
- Real-time filtering/search
- Responsive UI using Bootstrap 5
- Random turn generation system
